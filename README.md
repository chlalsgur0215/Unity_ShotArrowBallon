# Unity_ShotArrowBallon

사용 엔진 : Unity
사용 툴 : Unity Hub

활을 쏴서 풍선을 맞추는 게임.
마우스 클릭 후 드래그의 위치에 따라 활을 당기는 위력이 정해지며 화살이 날라가는 거리가 증가함.
화살은 포물선을 그리며 날아가게 되어있음.

연습모드 / 점수모드 / 커스텀 으로 구성되어있음
연습모드 에서는 과녁만 존재하며 활 사용의 연습이 가능함.
점수모드 에서는 랜덤한 위치에서 풍선이 나오고, 천천히 위로 올라가며 일정 위치까지 올라가게 된다면 사라지게 되어있음.
화살은 정지한 후 시간이 지나면 삭제되게끔 되어있음.
커스텀 모드에서 지정해놓은 활의 종류가 나오고, 클릭시 그 활로 변경이 가능함. (디폴트 활은 지정되어있음)
