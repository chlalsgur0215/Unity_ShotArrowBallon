# Unity_ShotArrowBallon

사용 엔진 : Unity
사용 툴 : Unity Hub

활을 쏴서 풍선을 맞추는 게임.
마우스 클릭 후 드래그의 위치에 따라 활을 당기는 위력이 정해지며 화살이 날라가는 거리가 증가함.
화살은 포물선을 그리며 날아가게 되어있음.

연습모드 / 점수모드 / 커스텀 으로 구성되어있음
연습모드 에서는 과녁만 존재하며 활 사용의 연습이 가능함.
점수모드 에서는 랜덤한 위치에서 풍선이 나오고, 천천히 위로 올라가며 일정 위치까지 올라가게 된다면 사라지게 되어있음.
화살은 정지한 후 시간이 지나면 삭제되게끔 되어있음.
커스텀 모드에서 지정해놓은 활의 종류가 나오고, 클릭시 그 활로 변경이 가능함. (디폴트 활은 지정되어있음)

이 에셋 스토어에서 필요 에셋들을 구해 변경해 사용하였고, 부가적으로 필요한 에셋은 만들어서 사용하였음.

보완하고싶은 점 / 추가하고 싶은 드
퀄리티 자체를 높이고싶음.
풍선이 좀 더 자연스럽게 올가가게 하고싶음. (속도를 조절해야 할 듯)
1인칭이 아닌, 3인칭으로 캐릭터 에셋의 애니메이션을 추가하여 활 시위를 당기는 모션을 추가해보고싶음.
활의 시위를 당길때 어디로 날라갈지 포물선 궤적이 보이게 해보고 싶음.
활 뿐 아니라 총, 마법 등 다양한 매개체를 이용하여 과녁을 맞추게 하고싶음.
화살을 날릴때 2개, 3개씩 날려서 더블샷, 트리플샷 같은 스킬을 추가하여보고싶음
리더보드같은 기능을 추가해 사용자간 경쟁이 가능한 시스템을 구축해보고싶음.
양궁처럼 과녁의 부위에 따라 점수를 부과해보고싶음.
